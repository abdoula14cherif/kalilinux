Installer un Logiciel sur Kali Linux

Guide complet pour installer des applications et outils sur votre système Kali Linux

Maîtrisez APT, le Gestionnaire de Paquets

Kali Linux utilise APT (Advanced Package Tool) pour gérer les logiciels. Apprenez à installer, mettre à jour et supprimer des applications en quelques commandes simples.

Terminal Kali Linux
root@kali:~# apt install nmap
root@kali:~# apt install wireshark metasploit-framework
root@kali:~# apt install
1

Ouvrir le Terminal

Pressez Ctrl + Alt + T ou cliquez sur l'icône terminal dans votre environnement de bureau. Toutes les installations se font depuis le terminal.

# Ouvrir le terminal avec raccourci clavier
Ctrl + Alt + T

Astuce

Vous pouvez aussi chercher "Terminal" dans le menu d'applications de votre bureau.

2

Tapez la Commande

Utilisez la commande apt install suivie du nom du logiciel que vous voulez installer. Vous devez avoir les droits administrateur (sudo).

# Structure de base de la commande
sudo apt install nom-du-logiciel

Exemple réel

Pour installer Firefox : sudo apt install firefox-esr

3

APT-GET Nom-du-Logiciel

L'ancienne commande apt-get fonctionne toujours, mais apt est recommandée. Elle télécharge et installe automatiquement le logiciel et ses dépendances.

# Version moderne (recommandée)
apt install nmap

# Ancienne version (fonctionne aussi)
apt-get install nmap

Note

Utilisez sudo si vous n'êtes pas connecté en tant que root.

Autres Méthodes d'Installation

Rechercher un Paquet

Cherchez un logiciel dans les dépôts avant de l'installer.

apt search nom-du-logiciel
apt-cache search mot-clé

Informations du Paquet

Obtenez des informations détaillées sur un paquet avant installation.

apt show nom-du-paquet

Installation Multiple

Installez plusieurs logiciels en une seule commande.

apt install logiciel1 logiciel2 logiciel3

Télécharger Sans Installer

Téléchargez le paquet .deb sans l'installer immédiatement.

apt download nom-du-paquet

Installation depuis .deb

Installez un paquet .deb téléchargé manuellement.

dpkg -i fichier.deb
apt install ./fichier.deb

Installation depuis Git

Installez directement depuis un dépôt Git.

git clone https://github.com/user/repo.git
cd repo && make && make install

Dépannage des Problèmes d'Installation

Erreur : "Unable to locate package"

Solution : Mettez à jour la liste des paquets : apt update. Vérifiez aussi l'orthographe du nom du paquet.

Erreur : "Failed to fetch" ou problèmes de réseau

Solution : Vérifiez votre connexion internet. Essayez de changer le miroir des dépôts dans /etc/apt/sources.list.

Erreur : "Dependencies problems"

Solution : Essayez apt --fix-broken install ou apt install -f pour réparer les dépendances.

Erreur : "Permission denied"

Solution : Utilisez sudo devant la commande ou connectez-vous en tant que root avec su -.

Installation Complète d'un Logiciel
user@kali:~$ sudo apt update
user@kali:~$ apt search hydra
user@kali:~$ apt show hydra
user@kali:~$ sudo apt install hydra -y
user@kali:~$ hydra --help
user@kali:~$
