Foire Aux Questions

Comment installer Kali Linux sur mon ordinateur ?

+

Pour installer Kali Linux, suivez ces étapes :

  • Téléchargez l'image ISO depuis le site officiel
  • Créez une clé USB bootable avec Rufus ou Etcher
  • Configurez le BIOS pour démarrer sur USB
  • Suivez l'assistant d'installation graphique
  • Sélectionnez le mode d'installation (graphique ou texte)
# Pour vérifier l'image téléchargée
sha256sum kali-linux-2023.4-installer-amd64.iso

Kali Linux ne détecte pas ma carte Wi-Fi, que faire ?

+

Si Kali Linux ne détecte pas votre carte Wi-Fi :

  • Vérifiez si le pilote est installé : lspci | grep Network
  • Installez les firmware requis : sudo apt install firmware-linux firmware-realtek
  • Activez les dépôts non-free : sudo nano /etc/apt/sources.list
  • Mettez à jour : sudo apt update && sudo apt full-upgrade
  • Redémarrez : sudo reboot

Comment mettre à jour Kali Linux ?

+

Pour mettre à jour Kali Linux proprement :

# Mettre à jour la liste des paquets
sudo apt update

# Mettre à jour les paquets installés
sudo apt upgrade

# Mise à jour complète (changements majeurs)
sudo apt full-upgrade

# Nettoyer les paquets inutiles
sudo apt autoremove
sudo apt autoclean

Conseil : Faites toujours un backup avant les mises à jour majeures.

Quels sont les outils essentiels pour débuter ?

+

Voici les outils essentiels pour débuter avec Kali Linux :

  • Nmap : Scanner réseau et ports
  • Wireshark : Analyse de trafic réseau
  • Metasploit : Framework de test d'intrusion
  • John the Ripper : Craqueur de mots de passe
  • Burp Suite : Test de sécurité web
  • SQLmap : Injection SQL automatisée
  • Aircrack-ng : Sécurité Wi-Fi
  • Hydra : Attaque par force brute

Comment réinitialiser le mot de passe root ?

+

Si vous avez oublié le mot de passe root :

  • Redémarrez Kali Linux
  • Appuyez sur ESC pendant le démarrage
  • Sélectionnez "Advanced options for Kali GNU/Linux"
  • Choisissez le mode "recovery mode"
  • Sélectionnez "root - Drop to root shell prompt"
  • Montez le système en écriture : mount -o remount,rw /
  • Changez le mot de passe : passwd root
  • Redémarrez : reboot

Kali Linux est-il légal à utiliser ?

+

Oui, Kali Linux est parfaitement légal. C'est une distribution Linux comme une autre.

Ce qui peut être illégal :

  • Utiliser Kali pour pirater des systèmes sans autorisation
  • Tester la sécurité de réseaux qui ne vous appartiennent pas
  • Violer des lois sur la protection des données

Utilisations légitimes :

  • Test de pénétration sur vos propres systèmes
  • Formation en cybersécurité
  • Audit de sécurité avec autorisation
  • Recherche en sécurité informatique

Conseil : Toujours obtenir une autorisation écrite avant de tester des systèmes externes.
